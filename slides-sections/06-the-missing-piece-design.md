# What lives in the gap

<v-clicks>

- **Modeling** — aggregates, entities, value objects
- **Responsibilities** — who does what, boundaries
- **Interfaces** — APIs, screens, contracts
- **Infrastructure** — storage, messaging, deployment
- **Test data** — realistic, representative
- **Enriched plan** — tasks decomposed from design decisions

</v-clicks>

<v-click>

<div class="mt-6 px-4 py-3 rounded-lg bg-blue-900/20 border border-blue-500/20 text-sm text-white/70">
  💡 AI can help with all of this — but it does not remove the need to <strong>think before you delegate</strong>.
</div>

</v-click>
